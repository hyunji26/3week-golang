# 02. Variables And Types

## 목표

- `var`와 `:=` 선언 방식을 구분합니다.
- Go의 기본 타입을 사용합니다.
- `fmt.Printf`로 값과 타입을 포맷팅합니다.

## 실행

```bash
go run ./02-variables-types
```

## 핵심 설명

- `var name string = "Alice"`는 타입을 명시하는 선언입니다.
    - 변수이름 + 변수타입 순서 주의할 것 !!
- `city := "Seoul"`은 타입을 추론하는 단축 선언입니다. 함수 내부에서만 사용할 수 있습니다.
- `:=`는 새 변수를 만들면서 값을 넣는 선언 연산자입니다. 왼쪽에 있는 변수 중 최소 하나는 새 변수여야 합니다.
- `=`는 이미 만들어진 변수에 새 값을 넣는 대입 연산자입니다. 아직 선언되지 않은 변수에는 사용할 수 없습니다.
    - 한 번 정해진 변수 타입은 바뀌지 않습니다. 예를 들어 `age := 25`로 선언하면 `age = 26`은 가능하지만 `age = "26"`은 불가능합니다.
- `%T`는 값의 타입을 출력하고, `%.1f`는 실수를 소수점 한 자리로 출력합니다.

## `:=`와 `=` 비교

```go
name := "Alice" // 새 변수 선언과 값 대입
name = "Bob"    // 기존 변수 값 변경

age = 25        // 에러: age가 아직 선언되지 않았음
age := 25       // 정상: age를 새로 선언
age = 26        // 정상: 기존 age 값 변경
```
