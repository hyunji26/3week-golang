# 01. Hello Go

## 목표

- Go 프로그램의 최소 구조를 이해합니다.
- `package main`, `import`, `func main()`의 역할을 설명할 수 있습니다.
- `go run`으로 프로그램을 실행합니다.

## 실행

```bash
go run ./01-hello
```

## 핵심 설명

- `package`는 현재 파일이 어떤 패키지에 속하는지 선언합니다. Go의 모든 `.go` 파일은 반드시 패키지 선언으로 시작합니다.
- `package main`은 실행 가능한 프로그램을 만들 때 사용하는 특별한 패키지 이름입니다. `go run`이나 `go build`로 실행 파일을 만들려면 `main` 패키지 안에 `main` 함수가 있어야 합니다.
- 다른 사람이 가져다 쓰는 라이브러리 코드는 보통 `package user`, `package calculator`처럼 기능 이름을 패키지명으로 사용합니다. 반면 직접 실행하는 프로그램의 시작 지점은 `package main`을 사용합니다.
- `import "fmt"`: 출력 기능을 제공하는 표준 라이브러리를 가져옵니다.
- `func`는 함수를 선언할 때 사용하는 키워드입니다.
- `func main()`은 프로그램의 시작점입니다. 운영체제가 Go 프로그램을 실행하면 Go 런타임이 `main` 패키지의 `main` 함수를 찾아 가장 먼저 실행합니다.
- `main` 함수는 매개변수와 반환값을 갖지 않습니다. 실행 결과를 외부에 알릴 때는 출력, 파일, 네트워크, 종료 코드 같은 방식을 사용합니다.
