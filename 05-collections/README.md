# 05. Collections

## 목표

- 배열과 슬라이스의 차이를 이해합니다.
- `append`로 슬라이스에 값을 추가합니다.
- 맵을 사용해 key-value 데이터를 저장하고 조회합니다.

## 실행

```bash
go run ./05-collections
```

## 핵심 설명

- 배열은 길이가 고정된 목록입니다. `[3]int{10, 20, 30}`은 `int` 값을 정확히 3개 담는 배열입니다.
- 배열은 길이가 타입의 일부입니다. `[3]int`와 `[4]int`는 서로 다른 타입입니다.
- 슬라이스는 길이가 고정되지 않은 목록입니다. `[]string{"Alice", "Bob"}`처럼 길이를 쓰지 않고 선언합니다.
- 슬라이스는 `append`로 값을 추가할 수 있습니다. `names = append(names, "Dave")`는 새 값이 추가된 슬라이스를 다시 `names`에 저장합니다.
- 맵 조회는 `value, ok := scores["Alice"]` 형태로 존재 여부를 함께 확인할 수 있습니다.

## 배열과 슬라이스 비교

| 구분 | 배열 | 슬라이스 |
| --- | --- | --- |
| 타입 표기 | `[3]int` | `[]int` |
| 길이 | 고정 | 변경 가능 |
| 길이가 타입에 포함되는가 | 포함됨 | 포함되지 않음 |
| 값 추가 | 직접 `append` 불가 | `append` 가능 |
| 주 사용처 | 길이가 정해진 데이터 | 일반적인 목록 데이터 |

```go
numbers := [3]int{10, 20, 30}       // 배열
names := []string{"Alice", "Bob"}   // 슬라이스

names = append(names, "Carol")      // 슬라이스는 값 추가 가능
```

강의에서는 우선 이렇게 기억하면 됩니다. 길이가 코드에서 중요한 고정 데이터면 배열, 일반적인 목록 처리는 슬라이스를 사용합니다.
